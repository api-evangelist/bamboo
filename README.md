# Bamboo Invest

Bamboo is an African investment platform providing REST APIs for fractional stock investing, dollar investments, portfolio management, and access to US and Nigerian capital markets. The platform enables fintech companies and financial services partners to embed brokerage and investment capabilities into their own applications.

## API

- **Documentation**: https://docs.investbamboo.com/
- **API Reference**: https://docs.investbamboo.com/api/
- **Help Center**: https://faq.investbamboo.com/
- **Blog / Learn**: https://learn.investbamboo.com/

## Key Capabilities

- User profile and brokerage account management
- US and Nigerian (NGX) stock trading with fractional share support
- Portfolio tracking (value, gains/losses, cost basis)
- Money movement: deposits and withdrawals
- Stock search and discovery
- Webhook-based settlement notifications
- KYC/identity integration
